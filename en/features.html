Features

— Windows shortcuts let you fly without leaving the keyboard.

— 01 / Editing & creation

Edit & create nodes

Put a node into edit mode, add new ones, extend branches, fold descendants — the most-used keys.

Key Action Description
F2 Edit node Instantly edit the selected node. F2 enters text editing in one keystroke instead of double-clicking.
Ctrl+Shift+N Create node Add one new node. If a node is selected, an empty one appears next to it. The first key to press.
Ctrl+Shift+A Multiple child nodes With a node selected, generate multiple empty children at once. For quick branching without AI.
Ctrl+Shift+W Create common parent With multiple nodes selected, creates a single common parent that wraps them as children.
Ctrl+Q Collapse / expand children Select a parent with children to hide all descendants (collapse). Press again to expand. Left-hand-only combo.
DEMO — F2 / Ctrl+Shift+N / Ctrl+Q for editing, creating, folding nodes
— 02 / Grouping

Group nodes

Range selectした複数ノードを 1 つにまとめたり、 付箋でゆるく束ねたり、 線をなぞって接続を一気に切ったり 大規模マップの構造を 自在に組み替える為のKey群です

Key Action Description
Shift+Drag Range select Hold Shift and drag to select all nodes in a rectangle. Then group, move, or recolor.
Ctrl+G Merge into one node Range selectしたノード群を merged into one node . The original nodes are bundled and treated as one. For drastic simplification.
Ctrl+Shift+G Group with sticky note Range selectしたノード群に 付箋名 to loosely bundle them. Individual nodes remain, but can be dragged or recolored together.に
Ctrl+Shift+V Toggle cut mode Switch to a mode where tracing connection lines cuts them. Press again to return. No more right-clicking each wrong line.
DEMO — Shift+Drag Range select / Ctrl+G 集約 / Ctrl+Shift+G 付箋 / Ctrl+Shift+V 裁断
— 03 / Cloud sync

Cloud sync

Sync to Firebase, refetch, or manage sync groups. One-press peace of mind at each milestone.

Key Action Description
Ctrl+S Cloud sync (upload) Syncs current map to Firebase, reflecting local edits. Press Enter to confirm dialog defaults.
Ctrl+D Cloud download (refetch) Refetches the cloud-saved map. To pull edits from another device. Becomes calendar event fetch in calendar mode.
Ctrl+Shift+S Open sync group Lists all maps in sync sync group panel. See at a glance which maps are up-to-date.
DEMO — Ctrl+S アッProード / Ctrl+D 取り直し / Ctrl+Shift+S sync group
— 04 / View switching

View switching

Call side menu, jump between maps, manage center on switching. For multi-map workflows.

Key Action Description
Ctrl+Shift+E Open side menu Toggle the side menu for map lists, folders, settings. No need to move the cursor to the edge.
Ctrl+1 〜 Ctrl+9 Jump to map (1-9) Jump to the 1st-9th maps in the drawer. Place favorites at the top for one-key switching.
Ctrl+B Set reference point Sets the reference point to the current view. Marks "this is where I want to view from".
Ctrl+Shift+B Go to reference point Ctrl+B で覚えさせたreference pointへ、 ビューを一気に戻します マップ内を 探索した後でも、 1 押しで「最初に決めた中心」 に帰還できます
DEMO — Ctrl+Shift+E サイドメニュー / Ctrl+1〜9 マップ移動 / Ctrl+B reference point / Ctrl+Shift+B 帰還
— 05 / Other features

Other features

Calculate, time, or check weather while mapping. Direct keys to built-in tools.

Key Action Description
Ctrl+Shift+T Light / dark mode One-press theme flip. Dark at night, light by day. Reduces eye strain.
Ctrl+Shift+C Open calendar Pops up the built-in calendar. Click a date to see schedules and map-linked events.
Ctrl+M Open calculator Instantly call the standard calculator. Basic math and %, for quick calcs.
Ctrl+Shift+M 高度Open calculator Scientific calculator mode. Trig, log, power, parentheses for complex calculations.
Ctrl+W Open weather Shows local weather based on location. Check without switching tabs.
Ctrl+T Stopwatch / timer Start elapsed time or countdown. For tracking work time or focus sessions.
Ctrl+P Pomodoro timer Dedicated Pomodoro: 25-min work + 5-min break. For long focus sessions.
DEMO — Ctrl+Shift+T theme / Ctrl+M calc / Ctrl+W weather / Ctrl+T stopwatch / Ctrl+P pomodoro

All shortcuts can be reassigned.

In the app, Settings → Shortcuts lets you change each key. Conflicts are auto-detected. Windows standard copy/paste/undo follow OS defaults in text fields. Shortcut requests via Contact directly to the developer.
