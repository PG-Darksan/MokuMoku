— 01 / Editing & creation

노드 편집・생성

선택 노드 편집 모드, 새 노드 추가, 가지 확장, 자손 접기 — 가장 자주 쓰는 키.

키 동작 개요
F2 노드 편집 선택 노드를 바로 편집 모드. 더블탭 대신 F2 한 번에 텍스트 편집.
Ctrl+Shift+N 노드 생성 새 노드 1 개 추가. 선택된 노드가 있으면 옆에 빈 노드가 나타남. 시작할 때 처음 누르는 키.
Ctrl+Shift+A 자식 노드 여러 개 생성 단일 노드 선택 후 누르면 개수를 지정해 빈 자식 노드를 일괄 생성. AI 없이 빠른 확장.
Ctrl+Shift+W 공통 부모 노드 생성 여러 노드 범위 선택 후 누르면 그것들을 자식으로 가지는 공통 부모 노드 1 개 생성.
Ctrl+Q 자식 노드 접기 / 펼치기 자식 있는 부모 선택 후 누르면 자손을 모아 숨김(격납). 다시 누르면 펼침. 왼손만으로 가능.
DEMO — F2 / Ctrl+Shift+N / Ctrl+Q 조합으로 노드 편집・생성・접기
— 02 / Grouping

노드 그룹화

범위 선택した複数ノードを 1 つにまとめたり、 付箋でゆるく束ねたり、 線をなぞって接続を一気に切ったり 大規模マップの構造を 自在に組み替える為の키群です

키 동작 개요
Shift+Drag 범위 선택 Shift 누르고 드래그로 사각 범위 노드 일괄 선택. 그룹화・이동・색상 변경.
Ctrl+G 하나의 노드로 합치기 범위 선택したノード群を 하나의 노드로 통합 . 원래 노드들은 묶여 단일 노드로 처리. 과감한 단순화에.
Ctrl+Shift+G 메모로 묶기 범위 선택したノード群に 付箋名 붙여 느슨하게 묶음. 개별 노드는 유지하면서 함께 이동/색상 변경 가능.に
Ctrl+Shift+V 절단 모드 전환 선 따라가 끊는 모드로 전환. 다시 누르면 복귀. 잘못된 선 우클릭 수고에서 해방.
DEMO — Shift+Drag 범위 선택 / Ctrl+G 集約 / Ctrl+Shift+G 付箋 / Ctrl+Shift+V 裁断
— 03 / Cloud sync

클라우드 동기화

Firebase 에 동기화, 다시 가져오기, 그룹 관리. 작업 매듭마다 한 번에 안심.

키 동작 개요
Ctrl+S 클라우드 동기화 (업로드) 현재 맵을 Firebase 에 동기화. 로컬 편집을 클라우드에 반영. Enter 로 기본 선택 확정.
Ctrl+D 클라우드 DL (다시 가져오기) 클라우드 맵 다시 가져오기. 다른 단말 편집 가져올 때. 캘린더 모드에서는 캘린더 이벤트 가져오기로.
Ctrl+Shift+S 동기화 그룹 열기 동기화 대상 맵을 목록으로 동기화 그룹 패널을 엽니다. 최신/미동기화 한눈에.
DEMO — Ctrl+S アッ프로ード / Ctrl+D 取り直し / Ctrl+Shift+S 동기화 그룹
— 04 / View switching

화면 전환

사이드 메뉴 호출, 맵 순간 이동, 전환 시 중심 관리. 여러 맵 작업자를 위한 키.

키 동작 개요
Ctrl+Shift+E 사이드 메뉴 열기 맵 목록・폴더・설정 사이드 메뉴 개폐. 마우스를 화면 가장자리로 옮길 필요 없음.
Ctrl+1 〜 Ctrl+9 맵 바로 이동 (1〜9) 드로어 상단부터 1〜9 번째 맵 순간 이동. 자주 쓰는 맵을 위에 두면 숫자 키 한 번에.
Ctrl+B 기준점 설정 맵 전환 시 중심이 되는 기준점 를 현재 뷰에 설정. "여기서 보고 싶다"는 거점을 기억.
Ctrl+Shift+B 기준점으로 이동 Ctrl+B で覚えさせた기준점へ、 ビューを一気に戻します マップ内を 探索した後でも、 1 押しで「最初に決めた中心」 に帰還できます
DEMO — Ctrl+Shift+E サイドメニュー / Ctrl+1〜9 マップ移動 / Ctrl+B 기준점 / Ctrl+Shift+B 帰還
— 05 / Other features

기타 기능

맵 작성하며 계산, 시간 측정, 날씨 확인. 내장 도구 직행 키.

키 동작 개요
Ctrl+Shift+T 라이트 / 다크 모드 전환 테마 한 번에 반전. 밤은 다크, 낮은 라이트. 눈 부담 경감.
Ctrl+Shift+C 캘린더 열기 내장 캘린더 팝업. 날짜 클릭으로 일정과 맵 연결 이벤트 확인.
Ctrl+M 계산기 열기 기본 계산기 즉시 호출. 사칙연산과 %, 빠른 계산.
Ctrl+Shift+M 高度계산기 열기 함수 계산기 모드. 삼각함수・로그・거듭제곱・괄호 등 복잡한 계산.
Ctrl+W 날씨 열기 위치 기반 현재 날씨 표시. 탭 전환 없이 확인.
Ctrl+T 스톱워치 / 타이머 경과 시간 측정/카운트다운 시작. 작업 시간 기록, 집중 시간 설정.
Ctrl+P 포모도로 타이머 전용 포모도로: 25 분 작업 + 5 분 휴식. 장시간 집중 작업 동반자.
DEMO — Ctrl+Shift+T 테마 / Ctrl+M 계산기 / Ctrl+W 날씨 / Ctrl+T 스톱워치 / Ctrl+P 포모도로
