Características

— Atajos Windows, sin soltar el teclado.

— 01 / Editing & creation

Editar y crear nodos

Pon el nodo en edición, añade nuevos, extiende ramas, pliega descendientes — las teclas más usadas.

Tecla Acción Descripción
F2 Editar nodo Edita el nodo al instante. F2 entra al texto en una sola pulsación.
Ctrl+Shift+N Crear nodo Añade un nodo. Si hay uno seleccionado, aparece otro vacío al lado. La primera tecla.
Ctrl+Shift+A Múltiples nodos hijos Con un nodo seleccionado, genera varios hijos vacíos. Para ramificar rápido sin IA.
Ctrl+Shift+W Crear nodo padre común Con varios nodos seleccionados, crea un padre común que los envuelve.
Ctrl+Q Plegar / desplegar hijos Selecciona un padre para ocultar descendientes (colapsar). Pulsa de nuevo para expandir.
DEMO — F2 / Ctrl+Shift+N / Ctrl+Q para editar, crear, plegar nodos
— 02 / Grouping

Agrupar nodos

Selección de rangoした複数ノードを 1 つにまとめたり、 付箋でゆるく束ねたり、 線をなぞって接続を一気に切ったり 大規模マップの構造を 自在に組み替える為のTecla群です

Tecla Acción Descripción
Shift+Drag Selección de rango Mantén Shift y arrastra para seleccionar nodos en rectángulo. Agrupa, mueve, recolorea.
Ctrl+G Fusionar en uno Selección de rangoしたノード群を fusionan en uno . Los nodos originales se tratan como uno. Para simplificación drástica.
Ctrl+Shift+G Agrupar con nota Selección de rangoしたノード群に 付箋名 se les pone una nota agrupándolos. Se mantienen separados pero se mueven juntos.に
Ctrl+Shift+V Modo corte Modo donde trazar líneas las corta. Pulsa de nuevo para volver. Sin clic derecho.
DEMO — Shift+Drag Selección de rango / Ctrl+G 集約 / Ctrl+Shift+G 付箋 / Ctrl+Shift+V 裁断
— 03 / Cloud sync

Sincr. en la nube

Sincroniza a Firebase, recupera, gestiona grupos. Tranquilidad en cada hito.

Tecla Acción Descripción
Ctrl+S Sincronización (subida) Sincroniza el mapa actual a Firebase. Pulsa Enter para confirmar diálogos.
Ctrl+D Descarga (refetch) Recupera el mapa de la nube. Para traer ediciones de otro dispositivo.
Ctrl+Shift+S Abrir grupo Lista los mapas sincronizados grupo panel. Ve de un vistazo qué está sincronizado.
DEMO — Ctrl+S アッProード / Ctrl+D 取り直し / Ctrl+Shift+S grupo
— 04 / View switching

Cambio de vista

Llama menú lateral, salta mapas, gestiona centro al cambiar. Para workflows multi-mapa.

Tecla Acción Descripción
Ctrl+Shift+E Abrir menú lateral Abre/cierra el menú para listas, carpetas, ajustes. Sin mover el mouse al borde.
Ctrl+1 〜 Ctrl+9 Saltar a mapa (1-9) Salta a los mapas 1-9 del cajón. Pon favoritos arriba para cambiar con una tecla.
Ctrl+B Establecer referencia Establece el punto de referencia en la vista actual. Marca "desde aquí quiero verlo".
Ctrl+Shift+B Ir a referencia Ctrl+B で覚えさせたpunto de referenciaへ、 ビューを一気に戻します マップ内を 探索した後でも、 1 押しで「最初に決めた中心」 に帰還できます
DEMO — Ctrl+Shift+E サイドメニュー / Ctrl+1〜9 マップ移動 / Ctrl+B punto de referencia / Ctrl+Shift+B 帰還
— 05 / Other features

Otras funciones

Calcula, mide tiempo, ve el clima mientras mapeas. Atajos a herramientas integradas.

Tecla Acción Descripción
Ctrl+Shift+T Modo claro / oscuro Tema invertido al pulsar. Oscuro de noche, claro de día. Menos fatiga visual.
Ctrl+Shift+C Abrir calendario Muestra el calendario integrado. Haz clic en una fecha para ver eventos.
Ctrl+M Abrir calculadora Llama la calculadora estándar al instante. Aritmética básica y %.
Ctrl+Shift+M 高度Abrir calculadora Calculadora científica. Trig, log, potencias, paréntesis.
Ctrl+W Abrir clima Muestra clima local según ubicación. Sin cambiar pestañas.
Ctrl+T Cronómetro / temporizador Inicia medición o cuenta regresiva. Registra tiempo o sesiones de enfoque.
Ctrl+P Temporizador Pomodoro Pomodoro: 25 min trabajo + 5 min descanso. Para sesiones largas.
DEMO — Ctrl+Shift+T tema / Ctrl+M calc / Ctrl+W clima / Ctrl+T cronómetro / Ctrl+P pomodoro

Todos los atajos se pueden reasignar.

En la app, Ajustes → Atajos permite cambiar cada tecla. Conflictos auto-detectados. Solicitudes de atajos vía Contacto al desarrollador.
