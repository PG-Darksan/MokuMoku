— 視聴は、 Sin distracciones

— 01 / YouTube Embed

URL を貼るだけで、 ノードに変身

YouTube の URL をコピペするだけで、 動画ノードが自動生成されます タイトル・サムネイルは oEmbed API で取得し、 即座に整理可能な状態に 通常動画はもちろん shorts や m.youtube.com 形式の URL にも対応

  • サムネ自動取得: YouTube CDN から hqdefault を即セット
  • shorts / m.youtube.com にも対応: モバイル向け URL もそのまま貼れる
  • タイトルも自動取得: oEmbed API でメタ情報を即補完
— 02 / Picture-in-Picture

画面の隅で、 ずっと再生されている

動画ノードをタップするとフルスクリーンへ、 もう一度小さくすると 画面隅の PiP プレイヤー として動き続けます マインドマップを 編集しながらでも視聴を継続

  • 左下ハンドルでリサイズ: 0.5x〜2.0x、 画面サイズに自動キャップ
  • ダブルタップで初期サイズに戻る: ドラッグでサイズが行き過ぎても安心
  • 中央タップで停止/再生: タップ操作とドラッグ移動が両立
  • 畳みFunción: ヘッダーだけ残して動画は裏で動かしっぱなし
Playback control

画面オフでも、 16 倍速でも、 鳴り続ける

— 01

バックグラウンド再生

audio_service ベースのフォアグラウンドサービスで、 アプリを閉じても音声を再生し続けます 通知欄から再生・一時停止操作も可

— 02

最大 16 倍速

WebView 経由で playbackRate を直接制御 講義動画やポッドキャストを高速消化したい人向け 一般プレイヤーの 2 倍を超える速度設定

— 03

視聴位置の永続化

アプリを再起動しても、 動画ごとの再生位置を覚えていてくれます 続きから視聴できるので、 章をまたぐような長尺動画でも快適

— 03 / Channel Queue

チャンネル全動画、 自動で連続再生

チャンネル URL を 1 つ与えると、 そのチャンネル内の全動画を自動で取得して キューに登録 視聴完了したら次の動画を自動補充するので、 関連動画への脱線を断ち切って 集中して見続けられます

  • 視聴完了したらキューから次の動画を自動補充
  • 動画ノードはマップ内に並ぶので、 進捗が一目瞭然
  • 関連動画取得 (右クリック) と組み合わせて、 興味の枝を自由に伸ばせる
— 04 / Related Videos

右クリックで、 リサーチが広がる

ノードを右クリック → 「関連動画を取得」 で、 そのノードのテキストを YouTube で検索した上位 5 件を子ノードとして配置 Teclaワードを起点に、 動画リサーチを枝分かれに展開できます

  • API Tecla不要 (公開 HTML をパース)
  • マップ内の既存動画とは重複しない
  • サムネイルも自動取得
  • YouTube ノード以外の テキストノード でも使える
