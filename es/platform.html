Plataforma

— PC とモバイル、 30 言語、 どこでも同じ体験を

— 01 / Multi-platform

PC とモバイル、 別物の UI で

同じFunciónを持ちながら、 操作系は環境ごとに最適化 PC は上部ヘッダー + Teclaボードショートカット、 モバイルは下部ボタンバー + 長押しメニュー 中途半端な「両対応」ではなく、 それぞれが本来の 使い方で完結します

  • Windows: 矢印Tecla長押しで連続移動、 Ctrl+1〜9 でフォルダー即移動
  • Android: 下部ボタンを長押しで差し替え・色変更・並び替え
  • SnackBar の自動退避: モバイルの下部 UI が通知に隠れないように
  • 画面オフ → 復帰時の状態保持: 編集中の状態を失わない
— 02 / 9 Languages

30 言語で、 母語のように

UI は 日本語 / 英語 / 中国語 / 韓国語 / スペイン語 / フランス語 / ドイツ語 / ポルトガル語 / ロシア語 / ヒンディー語 / アラビア語 / ベンガル語 / インドネシア語 / ウルドゥー語 / ナイジェリア・ピジン / エジプト・アラビア語 / マラーティー語 / ベトナム語 / テルグ語 / ハウサ語 / トルコ語 / 西パンジャブ語 / スワヒリ語 / タガログ語 / タミル語 / 広東語 / 呉語 / ペルシャ語 / タイ語 / ジャワ語 の 30 言語に対応 初回起動時の言語選択は海外展開を意識して 英語をデフォルト 表示しています

  • 初回起動フロー: 言語選択 → ユーザー名 (省略可) → ショートカット紹介 (PC のみ)
  • 翻訳Teclaは数千件: ボタン文言からエラーMensajeまで丁寧にローカライズ
  • AI 翻訳との連動: ノード内容も同じ 30 言語に翻訳可能
Organization

大きなマップを、 きれいに片付ける

— 01

付箋名グルーピング

複数のノードを 付箋名 でグループ化 ドラッグでグループ全体を 一度に動かしたり、 まとめて色を変えたり 大規模マップの整理に

— 02

容器ノード

関連するノード群を 容器ノード に格納すれば、 マップが一気に シンプルに 必要な時だけ右クリック → 「解除」で展開

— 03

方向ロック・スナップ

ノードの伸びる方向を 2 方向 / 4 方向 / 8 方向 で固定 拡大率や水平/垂直のロックも可能 整然としたレイアウトを保ちたい人向け

— 03 / Messaging

職人へ直接、 Mensaje

Reporte de error・Función要望は、 En la app, MensajeFunción から直接送信できます 開発者本人に届く為、 反映が早いのが特徴 送信時にユーザー名の設定が 必要です (匿名通報を防ぐ為)

  • カテゴリ選択 (バグ / Función要望 / Otro)
  • 過去の問い合わせ履歴と返信を確認可能
  • レート制限 (開発者モード時は免除)
